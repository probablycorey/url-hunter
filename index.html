URL Hunter!

An experimental game using the URL bar as the game screen.

Sorry Safari User

Safari doesn't show the full url in the URL bar, so this game won't work. To play try using Chrome or Firefox.

How to play?

Look up at the URL bar! You are the O! You are trying to hunt the a"s. Use the left and right keys to move. When you are over an a press spacebar to shoot!
